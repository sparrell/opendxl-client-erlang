Description

This sample demonstrates wrapping an existing service and exposing it on the DXL fabric.

In this particular case, the "openweathermap.org" current weather data is exposed as a DXL service. This service wrapper delegates to the OpenWeatherMap REST API.

openweather.hrl must be edited to include the OpenWeatherMap API key (see http://openweathermap.org/appid)
